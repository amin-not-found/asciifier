Use `--help` to get to know how to use the program:

```console
$ ./main.py --help
usage: asciifier [-h] [-W WIDTH] [-H HEIGHT] [-d] [-f {text}] filename

positional arguments:
  filename

options:
  -h, --help            show this help message and exit
  -W WIDTH, --width WIDTH
  -H HEIGHT, --height HEIGHT
  -d, --detailed
  -f {text}, --format {text}
```
